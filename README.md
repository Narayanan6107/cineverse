# CineVerse

CineVerse is a full-stack MERN application for discovering, tracking, and reviewing movies and TV series.

The platform uses the TMDB API to provide movie and TV show information while allowing users to maintain their own personalized account.

## Features

- User authentication
- Browse trending and popular movies & TV series
- Search content
- Filter by genre, language, rating, and release year
- View detailed information about movies and TV shows
- Add titles to a watchlist
- Mark content as watched
- Rate and review movies and TV series
- Manage favourite genres
- View watch history and profile
- Get personalized recommendations based on viewing history

## Tech Stack

- React
- Node.js
- Express.js
- MongoDB
- TMDB API
